-----------------------------
Запуск jobs'ов с парамметрами
-----------------------------

Парамметры - это по сути устновка переменных окружения при выполнении build'а, но
не всегда.

На скринштотах (screnshots-parameters) - все показано предельно ясно.
В качевте build'а мы указали скрипт который находится в файле - build-build-parameter.sh.

С помощью парамметра - file, выбрать файл и закачать его по пути, который указывается в парамметре.

С помощью парамметра - chose, можно выбрать значение переменной указаной в парамметре.

С помощью парамметра - string, можно указать переменную.
