------------------------------------------------
Deployment из GitHub репозитория в AWS Beanstalk
------------------------------------------------

Опишем тезисно т.к. почти все мы это уже делали.



На скриншотах показан весь процесс.

1 - 3 - Создали beanstalk application с PHP примером

Установили плагин на jenkins - https://plugins.jenkins.io/awseb-deployment-plugin/

Создали в AWS пользователя, которым будет пользоваться jenkins.

После установки плагина у нас будет возможность указать AWS credentionals.

4 - Создаем credentionals для jenkins'а

5 - 10 - Создаем job.
На скриншоте 10 можно увидеть, что вы указали версию. Это удобно. Эта версия нашего
приложения будет видна на скриншотах - 14 и 16

В GitHub репозитории делаем webhook - 0.

11 - 14 -  Делаем commit. Происходит deployment.

15 - 18 - Изменяем файлы в репозитории и делаем commit снова. Происходит deployment.

Ну и так далее.
